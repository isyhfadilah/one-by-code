Tentang One by Code

One by Code adalah unit kegiatan mahasiswa yang fokus pada pengembangan skill pemrograman dan teknologi. Kami menyelenggarakan berbagai kegiatan untuk meningkatkan kompetensi coding mahasiswa.

Visi & Misi

  • Menciptakan lingkungan belajar coding yang kolaboratif
  • Mempersiapkan mahasiswa untuk industri teknologi
  • Mengembangkan proyek-proyek open source yang bermanfaat
  • Menjadi hub inovasi teknologi di kampus
