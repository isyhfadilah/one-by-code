Tentang One by Code

One by Code adalah unit kegiatan mahasiswa yang fokus pada pengembangan skill pemrograman dan teknologi. Kami menyelenggarakan berbagai kegiatan untuk meningkatkan kompetensi coding mahasiswa.

Visi & Misi

  • Menciptakan lingkungan belajar coding yang kolaboratif
  • Mempersiapkan mahasiswa untuk industri teknologi
  • Mengembangkan proyek-proyek open source yang bermanfaat
  • Menjadi hub inovasi teknologi di kampus

Produk & Layanan Kami

Workshop & Training

Program pembelajaran intensif untuk berbagai bahasa pemrograman dan teknologi terkini mulai dari beginner hingga advanced level.

  • Web Development (HTML, CSS, JavaScript)
  • Backend Development (Node.js, Python, Java)
  • Mobile Development (Flutter, React Native)
  • Database & SQL

Mentoring & Konsultasi

Dapatkan bimbingan langsung dari praktisi dan mentor berpengalaman di bidang teknologi dan pemrograman.

  • 1-on-1 Code Review
  • Project Guidance
  • Career Consulting
  • Portfolio Development

Kompetisi & Challenge

Uji kemampuan coding Anda melalui berbagai kompetisi dan coding challenge yang kami selenggarakan sepanjang tahun.

  • Weekly Coding Challenge
  • Hackathon Event
  • Algorithm Competition
  • Bug Bounty Program

Kolaborasi Proyek

Bergabung dengan tim untuk mengembangkan proyek open source dan real-world application yang berdampak.

  • Open Source Projects
  • Client Projects
  • Internal Tools Development
  • Research & Development

Networking & Community

Bangun jaringan profesional dengan sesama developer, tech leaders, dan industry professionals.

  • Networking Events
  • Tech Talk Series
  • Community Forum
  • Industry Connections

Sertifikasi & Achievement

Raih sertifikat dan pengakuan untuk setiap pencapaian dan kontribusi Anda dalam komunitas kami.

  • Course Certification
  • Project Badges
  • Contribution Points
  • Achievement Records
